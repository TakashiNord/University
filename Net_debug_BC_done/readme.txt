Основная программа проекта для проекта  "Net_Owl_BC"
         для использования по предмету  "Мат. моделирование." 

Рабочий Проект :RGZ (Сетевая задача ' Общая Задача о максимальном потоке')
Текст программы. TurboС++.--------------------------------------(Borland)
Запуск: из среды  Turbo С++
 в ней настраиваются  
i=NetCalc("file1",
          "file2",
          "file3",
          "file4",
            int FlagMinMax ,int FlagCase,int FlagSpot,int FlagCalc,int MaxStep);
здесь :
   file1  -  файл с таблицей  ДЛИН (таблица = квадратная матрица (<>0)) {Формирование матрицы(пропускных способностей)r[i,j] из файла}
   file2  -  файл с таблицей  ЦЕН (таблица = квадратная матрица (<>0)) {Формирование матрицы(стоимости)c[i,j] из файла}
   file3  -  файл с таблицей  ИНТЕНСИВНСТЕЙ ВЕРШИН (строка или столбец (<>0)) {Формирование матрицы(интенсивностей)D[i] из файла}
   file4  -  файл рассчитанных  данных (таблица)  {Выходной файл рассчитанных данных=Оптимальный поток x[i,j] }
  FlagMinMax=1;{1- min ; 2- max ; (<-Тип решения Задачи)}
  FlagCase=2;{1- в порядке возрастания;2- в порядке убывания;3- сл. образом (<-Выбор вершины)}
  FlagSpot=1;{1- с начала;2- с конца(<-Расстановка пометок)}
  FlagCalc=1;{1-On;0-Off (Вычисление значение потока)}
  MaxStep=500;{(Аварийная остановка)Максимальное число проходов для нахождения Потока}

Возвращает 
  -1:Cannot read data files;
  -2 :Cannot creat out file;
   0  :all normal
