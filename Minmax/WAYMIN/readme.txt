Основная программа проекта
Программа рассчитывает 'Оптимальное суммарное значение', 'Путь по вершинам'  и создает таблицу расстояний.
         для использования по предмету  "Мат. моделирование." 

Проект  "WAYMIN.TXT"
Среда  "Turbo Pascal 7.0 (Borland)" 
 написан в 2000 году.

Запуск: из среды  Turbo Pascal

 в ней настраиваются  
   file1  -  файл с исходной таблицей (таблица = квадратная матрица (<>0))
   file2  -  файл рассчитанных  данных (таблица)
   1      -  расчет минимального расстояния
или 
   2      -  расчет максимального расстояния
+
и остальные параметры
